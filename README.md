A simple web application to calculate and analyze your Body Mass Index (BMI).

## Features ✨

- BMI Calculator
- Health Category Classification
- Input Validation
- Responsive Design
- Unit Tested

## Development 👨‍💻

```bash
# Install dependencies
npm install

# Run tests
npm test

# Run test coverage
npm run test:coverage

# Start development server
npm start
```

## Tech Stack 🛠️

- HTML5
- CSS3
- JavaScript
- Jest (Testing)
